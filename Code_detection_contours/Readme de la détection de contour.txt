Readme de la détection de contour

!Recommandation importante!

1) Lancez le script depuis Anaconda Prompt (Windows) plutôt que depuis un IDE classique.

2) Les deux images doivent avoir la même résolution et les mêmes dimensions.

3) N'hésitez pas à modifier le pas de rotation directement dans le code (ligne 194).

Étapes :
1) Lancer le code.

2) Sélectionner le centre du disque de lecture, puis fermer la fenêtre.

3) Sélectionner le centre du disque de forme, puis manipuler la tracebar à votre convenance (conseil : seuil min = 255, seuil max = 0).

4) Sélectionner les formes souhaitées (attention : une fenêtre s'ouvre à chaque sélection, n'hésitez pas à les fermer).

5) Valider avec Entrée pour lancer la rotation et les calculs de surface d'intersection.

6)Récupérer le fichier Excel (intersections_bob_rotation.xlsx) dans le même dossier que votre programme.
